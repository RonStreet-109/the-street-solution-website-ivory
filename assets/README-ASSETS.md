# Graphics Assets

These assets are included so the website runs immediately after upload.

- `ts-logo.png` — cropped working logo from the approved visual reference. Replace with the final transparent PNG/SVG logo when ready.
- `hero-books.png` — cropped working hero image from the approved visual reference. Replace with the final high-resolution hero image if needed.
- `ron-street.png` — cropped working footer portrait from the approved visual reference. Replace with the final approved headshot.
- `botanical-line.svg` — decorative line-art accent for the note panel.
- `approved-homepage-reference.png` — the approved homepage reference image for comparison/quality control.
- `hero-books-placeholder.svg` — optional vector placeholder if you need a non-photo hero image later.
- `favicon.svg` — simple temporary browser icon.

Keep the same filenames to avoid changing code, or update the image paths in `index.html`.
